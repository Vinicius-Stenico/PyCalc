# Contribuição

Contribuições são bem-vindas para correções de bugs, melhorias de documentação, testes, sugestões de interface e validação de builds.

Este projeto não é open source. Ao enviar uma issue, sugestão ou pull request, você concorda que sua contribuição poderá ser analisada, adaptada e incorporada ao PyCalc pelo autor do projeto, sem que isso conceda permissão para copiar, vender, redistribuir ou reutilizar partes significativas do código sem autorização prévia.

Contribuições aceitas poderão receber créditos no README, em releases ou no arquivo CONTRIBUTORS.md.
